# Commit AS — Teknisk arkitekturbeslutning

## ADR-0042: Valg av sky-plattform for AI-tjenester

### Status
Godkjent — 2026-02-15

### Kontekst
Vi trenger en sky-plattform som støtter AI-inferens med EU data residency 
for kunder med strenge personvernkrav.

### Beslutning
Vi velger AWS Bedrock med EU-inferensprofiler som primærplattform.

### Begrunnelse
- Claude Opus 4.6 er tilgjengelig via Bedrock med EU-ruting
- AWS har datasentre i Stockholm og Frankfurt
- Bedrock integrerer med eksisterende IAM-oppsett
- Pay-per-token-modell passer vårt bruksmønster

### Konsekvenser
- ~10% prisøkning for EU-regional inferens
- Utviklere må konfigurere AWS credentials
- Vi må vedlikeholde DPA med AWS
